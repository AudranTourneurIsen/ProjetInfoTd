Polygon TD is a end-year project for six students in an informatic school. We had to do a reflection game only and we chose to adapt the playstyle of a Tower Defense.

Team :

  • Antoine
  • Audran
  • Emilie
  • Gabin
  • Nicolas
  • Théodore

But, why a tower defense ?

We wanted to do a fun game even if the subject is based on a reflection game.
We were thinking about some games but we couldn't add a solver, and this is required for our project. So we just thought a little longer. And then, we finally thought about a Tower Defense based game.

We had 3 weeks to create it with only C, CSS, HTML et JavaScript languages, what a ride !

This is how Polygon TD is born.
